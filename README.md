# API Quotes Generator
## Introduction
This project generates random quotes using the API provided by Frontend Mentor. Users can click on the dice to generate a new quote. The project is built using React, Vite, and Tailwind CSS.

## Installation
Clone this repository to your local machine.
Run npm install in the root directory of the project to install the necessary dependencies.
Run npm run dev to start the development server.
Navigate to localhost:3000 in your browser to view the project.
## Usage
Click on the dice button to generate a new quote.
